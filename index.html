PROBLEM STATEMENT

In most of college campuses, there are several ways to get in and go out. So, if we use the traditional way of inserting entries, it will definitely be hectic to sync all the gates among themselves. Thus, to prevent these additional extra efforts and to strengthen the security measures, it is better to digitalize the whole process.

PROJECT DESCRIPTION

In this project, I am using basic HTML, CSS to make the frontend. PHP is choosen as backend programming language which will work with MySQL database. We are gonna provide basic features of recording out time and in time, along with the feature to update the database of students, and enquire the values of tuples according to need.
